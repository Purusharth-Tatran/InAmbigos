# 🌍 InAmigos Foundation – NGO Awareness Webpage

## 📖 Project Overview

This project is a responsive NGO Awareness Webpage developed using **HTML5** and **CSS3**. It aims to spread awareness about the initiatives and social impact of **InAmigos Foundation** by presenting information in a clean and attractive user interface.

The webpage highlights the organization's mission, ongoing projects, achievements, and encourages visitors to volunteer or support the NGO.

---

## 🚀 Features

- Responsive Navigation Bar
- Hero Image Slider
- About InAmigos Foundation
- Ongoing Projects Section
- Social Impact Statistics
- Call-to-Action Section
- Attractive Cards with Hover Effects
- Responsive Design
- Modern UI with Animations

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome Icons

## ▶️ How to Run

1. Download or clone the project.
2. Place all images inside the **images** folder.
3. Open **index.html** in any modern web browser.

No additional software or server is required.
